# Research: Kraken Weird Relations

## Objective
Investigate the "weird relation" reported between:
- WBTC/USD
- BTC/USD
- TBTC/USD (User said TBCT, likely TBTC)
- XAUT/USD
- LSETH/USD
- TAO/USD

## Steps
1. [ ] **Symbol Verification**: Confirm ticker symbols on Kraken.
2. [ ] **Data Acquisition**: Fetch 1h and 5m data for all assets.
3. [ ] **Correlation Analysis**: Compute Pearson and Spearman correlations.
4. [ ] **Spread Analysis**: continuous spread monitoring between pegged assets (BTC variants).
5. [ ] **Report**: Generate a heatmap and z-score plot.
